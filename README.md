# WiFi 小车控制面板

1. 把这三个文件放到同一目录；
2. 在该目录打开命令行：

   ```bash
   python -m http.server 8000
   ```

3. 浏览器打开：

   ```text
   http://localhost:8000
   ```
4. 在 **Broker URL** 填入你的 MQTT WebSocket 地址（如 `ws://192.168.1.100:8083/mqtt`），点击 **连接**；
5. 连接成功后，使用以下方式控制小车：
   - **键盘控制**：方向键 ↑↓←→ 或 WASD 键
   - **鼠标控制**：拖拽虚拟摇杆
   - **触摸控制**：在移动设备上可以触摸拖拽摇杆
   - **Xbox手柄控制**：使用左摇杆控制小车移动
6. 开始接收 `car/status` 上报的数据，状态面板会实时刷新；
7. 虚拟摇杆会实时显示当前的控制状态和输出值。
